# IT-Support-Lab-Scenarios
Author: Bailey Popovic

Last Updated: May 2026


This is my personal IT support lab portfolio. Each lab was documented by me to prepare for service desk, desktop support, and security engineer roles.

The repository contains 15 labs covering the skills that appear most consistently in IT support and help desk job descriptions. Every lab folder holds the full written walkthrough in a README.md file. At the end of each README, there is a Troubleshooting Reference table for quick viewing of troubleshooting steps.
